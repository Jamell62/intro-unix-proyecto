#Intro Unix Project

#Project Description
This project was created to apply what we learned about UNIX/Linux systems, basic terminal usage, version control with Git, and digital signatures using GPG.  
The main idea is to understand how to manage files, track changes in a project, and ensure that all contributions are authentic and secure.
